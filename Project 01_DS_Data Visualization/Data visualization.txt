Bar Chart
A bar chart is used to display the distribution of a categorical variable. For example, let’s visualize the distribution of genders in a population.

Steps:
1.Prepare the Data: Create a dataset with the categorical variable. 
  2.Import Libraries: Use libraries like matplotlib or seaborn. 
  3.Create the Bar Chart: Use the bar function from matplotlib

Histogram
A histogram is used to display the distribution of a continuous variable. For example, let’s visualize the distribution of ages in a population.

Steps:
1.Prepare the Data: Create a dataset with the continuous variable.
2.Import Libraries: Use libraries like matplotlib or seaborn. 
3.Create the Histogram: Use the hist function from matplotlib
